## 🚀 Installation Guide

1. Clone the repository:
   ```bash
   git clone https://github.com/AungPyaeSonOo/React-Note-Pack-Project.git
   ```
2. Navigate into the project folder:
   ```bash
   cd React-Note-Pack-Project
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the project:
   ```bash
   npm start
   ```
